



-DO NOT replace tabs with spaces
-opening braces are on their own line
-all identifiers are in camelCase
-Class, enum, struct type names start with a capital
-A struct should be used exactly as if it were a C-style struct.
-Never write "using namespace ..." in a .h file.
-Writing "using namespace NAME" in a .cpp file is only allowed for files implementing classes and functions declared inside that namespace NAME.
-There is one-to-one correspondence between folders and namespaces.
